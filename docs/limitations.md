# Limitations

- This repository is a **paper artifact package**, not an end-to-end operational forecasting system.
- Public data are constrained to monthly aggregates and do not expose raw Dataset A/B records.
- The included outputs are fixed artifact files intended for ICCE reviewer verification.
- Supporting files such as `results/tables/paper_prophet_summary.csv` and `results/tables/figure3_prophet_forecast_data.csv` are contextual aids; the main paper result mapping remains Figure 1/Figure 2/Figure 3/Table II/Table III/Table IV.
