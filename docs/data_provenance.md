# Data Provenance

This artifact package uses three monthly sources aligned with the paper narrative.

## Source groups and public mapping
- **Dataset A** (monthly aggregated real-estate transaction series)
  - Public file: `data/public/dataset_a_monthly_public.csv`
- **Dataset B** (monthly aggregated land-registration activity counts)
  - Public file: `data/public/dataset_b_monthly_public.csv`
- **Dataset C** (monthly macroeconomic indicators)
  - Public file: `data/public/dataset_c_macro_public.csv`

A merged modeling panel is provided at:
- `data/public/unified_monthly_panel_public.csv`

## Release boundary
Raw source-level records for Dataset A and Dataset B are excluded; only anonymized monthly aggregates are released for reviewer checks.
